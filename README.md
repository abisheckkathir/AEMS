# To install modules:
#### - Run yarn install
# To run server: 
#### - Run yarn run server

# To run the client:
#### - Run yarn run client

# To run integration testing:
#### - Install Selenium on Python using "pip install selenium"
#### - Download the webdriver
#### - Add the webdriver to path
#### - Run python integrationtesting.py
